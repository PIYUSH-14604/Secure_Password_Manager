====================================
   🔐 SECURE PASSWORD MANAGER 🔐
====================================

A simple and secure Password Manager built using Python.  
This project helps users safely store, manage, and retrieve passwords while learning the fundamentals of cybersecurity, encryption, and Python development.

---

## 🚀 Features

- Store passwords securely
- Retrieve saved credentials
- Simple and beginner-friendly UI/CLI
- Local password management
- Lightweight and fast
- Organized credential handling
- Python-based implementation

---

## 🛠️ Tech Stack

- Python
- File Handling
- Encryption / Security Concepts
- CLI (Command Line Interface)

---

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/PIYUSH-14604/Secure_Password_Manager.git
```

### 2️⃣ Navigate to project directory

```bash
cd Secure_Password_Manager
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the application:

```bash
python main.py
```

Follow the on-screen instructions to:
- Add passwords
- Save credentials
- Retrieve stored passwords
